MapBound

I contributed to the conceptualization of a GPS tracking system tailored for police officers, with a focus on precise location monitoring and heightened security measures. This envisioned system seamlessly transmits real-time location data to a centralized server, enabling dynamic display, geofencing, and proactive notifications. Leveraging an object detection model, it distinguishes trace paths from vacant areas on maps, while LoRa transmitters ensure continuous tracking even in low-network areas. This conceptual framework provides law enforcement agencies with advanced surveillance capabilities and valuable insights into security trends.

I played a key role in conceptualizing CertifyiT, a forward-thinking certificate management system designed to revolutionize traditional certification processes. By integrating Arweave's Blockweave and Solana's blockchain, CertifyiT ensures unparalleled security and transparency. With features such as automated certificate generation and user-friendly verification, CertifyiT promises to streamline certification procedures. It offers a Practical, Robust, Secure, and Decentralized Method for Certificate Generation and Validation along with Digital File Storage.
